## Instalação única

```bash
curl -fsSL https://raw.githubusercontent.com/wesleysimplicio/simplicio/master/install.sh | sh
```

O install.sh:
1. Detecta OS + arquitetura
2. Baixa o binário simplicio da GitHub release
3. Instala simplicio-agent com pip (com suporte a voz)
4. Instala pvporcupine + sounddevice (wake word)
5. Cria ~/.simplicio/tami-config.json com Tami ativa por padrão
6. Exibe mensagem de boas-vindas
