# Simplicio Agent

**La velocidad de Hermes. El determinismo de Simplicio. Entrega verificada.**

Simplicio Agent es un fork público e independiente del [Hermes Agent oficial](https://github.com/NousResearch/hermes-agent). Conserva sus herramientas, proveedores, skills, canales y aprendizaje, y añade Simplicio Runtime en Rust: control de acciones, checkpoints y deshacer, edición mecánica, bucle basado en evidencias y recibos HBP.

- ⚡ rutas rápidas medidas y daemon caliente;
- 🛡️ efectos gobernados por un runtime compilado;
- ✅ finalización solo después de validar y recomputar;
- 🧠 memoria local con los guardianes Isa, Helo y Levi;
- 🏠 ejecución local-first y datos del usuario;
- 🌐 CLI, MCP, escritorio, mensajería, plugins y múltiples modelos.

Consulta el [README canónico en inglés](../README.md) para las imágenes, Mermaid, benchmarks reproducibles, comparación completa, instalación y Star History.
