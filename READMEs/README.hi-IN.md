# Simplicio Agent

**Hermes की गति। Simplicio का निर्धारणवाद। प्रमाणित डिलीवरी।**

Simplicio Agent [आधिकारिक Hermes Agent](https://github.com/NousResearch/hermes-agent) का सार्वजनिक और स्वतंत्र fork है। यह tools, providers, skills, gateways और learning को बनाए रखता है तथा Rust में Simplicio Runtime जोड़ता है: action gate, checkpoint/undo, mechanical editing, evidence-gated loop और HBP receipts।

- ⚡ मापे गए fast paths और warm daemon;
- 🛡️ compiled runtime द्वारा नियंत्रित effects;
- ✅ validation और recomputation के बाद ही completion;
- 🧠 Isa, Helo और Levi के साथ local memory;
- 🏠 local-first, user-owned data;
- 🌐 CLI, MCP, desktop, messaging, plugins और multiple models।

चित्र, Mermaid, benchmarks, पूर्ण तुलना, installation और Star History के लिए [मुख्य README](../README.md) देखें।
