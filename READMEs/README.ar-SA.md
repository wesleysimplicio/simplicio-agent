# Simplicio Agent

**سرعة Hermes. حتمية Simplicio. تسليم مُثبت.**

Simplicio Agent هو تفرع عام ومستقل من [Hermes Agent الرسمي](https://github.com/NousResearch/hermes-agent). يحتفظ بالأدوات والمزودين والمهارات والبوابات والتعلّم، ويضيف Simplicio Runtime المكتوب بـ Rust: بوابة إجراءات، نقاط استعادة وتراجع، تحرير ميكانيكي، حلقة قائمة على الأدلة وإيصالات HBP.

- ⚡ مسارات سريعة مقاسة وdaemon دافئ؛
- 🛡️ تأثيرات يحكمها runtime مُجمّع؛
- ✅ اكتمال بعد التحقق وإعادة الحساب فقط؛
- 🧠 ذاكرة محلية مع Isa وHelo وLevi؛
- 🏠 تشغيل local-first وبيانات يملكها المستخدم؛
- 🌐 CLI وMCP وسطح المكتب والمراسلة والإضافات ونماذج متعددة.

راجع [README الإنجليزي الأساسي](../README.md) للصور وMermaid والاختبارات والمقارنة الكاملة والتثبيت وStar History.
