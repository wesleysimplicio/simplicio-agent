# Simplicio Agent

**Szybkość Hermesa. Determinizm Simplicio. Zweryfikowane dostarczenie.**

Simplicio Agent to publiczny, niezależny fork [oficjalnego Hermes Agent](https://github.com/NousResearch/hermes-agent). Zachowuje narzędzia, dostawców, umiejętności, bramy i uczenie, dodając Simplicio Runtime w Rust: kontrolę działań, checkpointy/undo, mechaniczne edycje, pętlę opartą na dowodach i pokwitowania HBP.

- ⚡ mierzone szybkie ścieżki i ciepły daemon;
- 🛡️ efekty kontrolowane przez skompilowany runtime;
- ✅ ukończenie dopiero po walidacji i ponownym obliczeniu;
- 🧠 lokalna pamięć Isa, Helo i Levi;
- 🏠 local-first i dane użytkownika;
- 🌐 CLI, MCP, desktop, komunikatory, pluginy i wiele modeli.

Grafiki, Mermaid, benchmarki, pełne porównanie, instalacja i Star History są w [głównym README](../README.md).
