# Simplicio Agent

**Velocidade do Hermes. Determinismo do Simplicio. Entrega comprovada.**

O Simplicio Agent é um fork público e independente do [Hermes Agent oficial](https://github.com/NousResearch/hermes-agent). Ele preserva ferramentas, provedores, skills, gateways e aprendizado do Hermes, acrescentando o Simplicio Runtime em Rust: gate de ações, checkpoints/undo, edição mecânica sem LLM, loop com evidências e recibos HBP.

- ⚡ caminhos rápidos medidos e daemon aquecido;
- 🛡️ efeitos controlados por runtime compilado;
- ✅ conclusão somente depois de validação e recomputação;
- 🧠 memória local e guardiões Isa, Helo e Levi;
- 🏠 execução local-first e estado pertencente ao usuário;
- 🌐 CLI, MCP, desktop, mensageria, plugins e múltiplos modelos.

Veja o [README canônico em inglês](../README.md) para as três imagens, todos os diagramas Mermaid, benchmark reproduzível, comparação item a item, instalação e Star History.

```bash
curl -fsSL https://simpleti.com.br/simplicio/install.sh | sh
simplicio-agent setup
simplicio-agent doctor
```
