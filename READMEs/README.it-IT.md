# Simplicio Agent

**Velocità di Hermes. Determinismo di Simplicio. Consegna verificata.**

Simplicio Agent è un fork pubblico e indipendente dell’[Hermes Agent ufficiale](https://github.com/NousResearch/hermes-agent). Mantiene strumenti, provider, skill, gateway e apprendimento, aggiungendo Simplicio Runtime in Rust: action gate, checkpoint/undo, modifiche meccaniche, loop basato su prove e ricevute HBP.

- ⚡ percorsi rapidi misurati e daemon caldo;
- 🛡️ effetti governati da un runtime compilato;
- ✅ completamento solo dopo validazione e ricalcolo;
- 🧠 memoria locale con i guardiani Isa, Helo e Levi;
- 🏠 esecuzione local-first e dati dell’utente;
- 🌐 CLI, MCP, desktop, messaggistica, plugin e più modelli.

Consulta il [README canonico in inglese](../README.md) per immagini, Mermaid, benchmark, confronto completo, installazione e Star History.
