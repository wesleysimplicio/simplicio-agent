# Simplicio Agent

**Hermes의 속도. Simplicio의 결정성. 검증된 전달.**

Simplicio Agent는 [공식 Hermes Agent](https://github.com/NousResearch/hermes-agent)의 공개 독립 포크입니다. 도구, 제공자, 스킬, 게이트웨이와 학습 기능을 유지하면서 Rust 기반 Simplicio Runtime의 액션 게이트, 체크포인트/실행 취소, 기계적 편집, 증거 기반 루프와 HBP 영수증을 추가합니다.

- ⚡ 측정된 고속 경로와 웜 데몬
- 🛡️ 컴파일된 Runtime이 제어하는 효과
- ✅ 검증과 재계산 이후에만 완료
- 🧠 Isa, Helo, Levi 로컬 메모리 가디언
- 🏠 로컬 우선 및 사용자 소유 데이터
- 🌐 CLI, MCP, 데스크톱, 메시징, 플러그인, 다중 모델

이미지, Mermaid, 재현 가능한 벤치마크, 전체 비교, 설치 및 Star History는 [영문 README](../README.md)를 참조하세요.
