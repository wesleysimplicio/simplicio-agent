# Simplicio Agent

**Hermes の速度。Simplicio の決定性。検証されたデリバリー。**

Simplicio Agent は[公式 Hermes Agent](https://github.com/NousResearch/hermes-agent)の公開・独立フォークです。ツール、プロバイダー、スキル、ゲートウェイ、学習機能を継承し、Rust 製 Simplicio Runtime によるアクションゲート、チェックポイント／Undo、機械的編集、証拠ベースのループ、HBP レシートを追加します。

- ⚡ 計測済み高速パスとウォームデーモン
- 🛡️ コンパイル済み Runtime による副作用の統制
- ✅ 検証と再計算後のみ完了
- 🧠 Isa・Helo・Levi によるローカルメモリ
- 🏠 ローカルファースト、ユーザー所有のデータ
- 🌐 CLI、MCP、デスクトップ、メッセージング、プラグイン、複数モデル

画像、Mermaid、ベンチマーク、全比較、導入方法、Star History は[英語版 README](../README.md)をご覧ください。
