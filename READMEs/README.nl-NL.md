# Simplicio Agent

**Hermes-snelheid. Simplicio-determinisme. Bewezen levering.**

Simplicio Agent is een openbare, onafhankelijke fork van de [officiële Hermes Agent](https://github.com/NousResearch/hermes-agent). Het behoudt tools, providers, skills, gateways en leren, en voegt de Simplicio Runtime in Rust toe: action gates, checkpoints/undo, mechanische edits, evidence-gated loops en HBP-receipts.

- ⚡ gemeten snelle paden en warm daemon;
- 🛡️ effecten bestuurd door een gecompileerde runtime;
- ✅ pas klaar na validatie en herberekening;
- 🧠 lokaal geheugen met Isa, Helo en Levi;
- 🏠 local-first en data in eigendom van de gebruiker;
- 🌐 CLI, MCP, desktop, messaging, plugins en meerdere modellen.

Zie het [canonieke Engelse README](../README.md) voor beelden, Mermaid, benchmarks, volledige vergelijking, installatie en Star History.
