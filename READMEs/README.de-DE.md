# Simplicio Agent

**Hermes-Geschwindigkeit. Simplicio-Determinismus. Nachgewiesene Auslieferung.**

Simplicio Agent ist ein öffentlicher, unabhängiger Fork des [offiziellen Hermes Agent](https://github.com/NousResearch/hermes-agent). Werkzeuge, Provider, Skills, Gateways und Lernen bleiben erhalten; hinzu kommt die Simplicio Runtime in Rust mit Action Gate, Checkpoints/Undo, mechanischen Änderungen, evidenzbasierter Schleife und HBP-Belegen.

- ⚡ gemessene Fast Paths und Warm-Daemon;
- 🛡️ kontrollierte Effekte durch kompilierte Runtime;
- ✅ Abschluss erst nach Validierung und Neuberechnung;
- 🧠 lokaler Speicher mit Isa, Helo und Levi;
- 🏠 Local-first und benutzereigene Daten;
- 🌐 CLI, MCP, Desktop, Messaging, Plugins und freie Modellwahl.

Alle Bilder, Mermaid-Diagramme, Benchmarks, Unterschiede, Installation und Star History stehen im [kanonischen englischen README](../README.md).
