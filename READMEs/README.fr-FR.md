# Simplicio Agent

**La vitesse d’Hermes. Le déterminisme de Simplicio. Une livraison vérifiée.**

Simplicio Agent est un fork public et indépendant de [Hermes Agent](https://github.com/NousResearch/hermes-agent). Il conserve outils, fournisseurs, skills, passerelles et apprentissage, puis ajoute le Runtime Simplicio en Rust : contrôle des actions, checkpoints/annulation, édition mécanique, boucle fondée sur les preuves et reçus HBP.

- ⚡ chemins rapides mesurés et daemon préchauffé ;
- 🛡️ effets gouvernés par un runtime compilé ;
- ✅ fin uniquement après validation et recomputation ;
- 🧠 mémoire locale et gardiens Isa, Helo et Levi ;
- 🏠 exécution local-first et données sous votre contrôle ;
- 🌐 CLI, MCP, desktop, messagerie, plugins et plusieurs modèles.

Voir le [README canonique en anglais](../README.md) pour les visuels, Mermaid, benchmarks, comparaison complète, installation et Star History.
