# Simplicio Agent

**Hermes 的速度，Simplicio 的确定性，可验证的交付。**

Simplicio Agent 是[官方 Hermes Agent](https://github.com/NousResearch/hermes-agent)的公开独立分支。它保留工具、模型供应商、技能、消息网关与学习能力，并加入 Rust 编写的 Simplicio Runtime：动作门控、检查点/撤销、机械式编辑、证据驱动循环和 HBP 回执。

- ⚡ 经测量的快速路径与常驻 daemon；
- 🛡️ 由编译 Runtime 管理副作用；
- ✅ 仅在验证与独立重算后完成；
- 🧠 Isa、Helo、Levi 本地记忆守护者；
- 🏠 本地优先，数据归用户所有；
- 🌐 CLI、MCP、桌面、消息、插件与多模型支持。

图片、Mermaid、可复现基准、完整对比、安装方式与 Star History 请查看[英文主 README](../README.md)。
