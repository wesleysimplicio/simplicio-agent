# Simplicio Agent

**Скорость Hermes. Детерминизм Simplicio. Проверенная доставка.**

Simplicio Agent — публичный независимый форк [официального Hermes Agent](https://github.com/NousResearch/hermes-agent). Он сохраняет инструменты, провайдеры, навыки, шлюзы и обучение, добавляя Simplicio Runtime на Rust: контроль действий, checkpoints/undo, механические правки, цикл с доказательствами и HBP-квитанции.

- ⚡ измеренные быстрые пути и тёплый daemon;
- 🛡️ эффекты под контролем скомпилированного runtime;
- ✅ завершение только после проверки и пересчёта;
- 🧠 локальная память с Isa, Helo и Levi;
- 🏠 local-first и данные пользователя;
- 🌐 CLI, MCP, desktop, сообщения, плагины и разные модели.

Изображения, Mermaid, тесты, полное сравнение, установка и Star History — в [каноническом README](../README.md).
